---
title: Github 上 emoji 的表情含义
date: 2019-08-14 16:13:50
tags: Git
---

-   🎨 - 改进结构和代码格式
-   ⚡️ - 优化性能
-   🔥 - 移除代码或文件
-   🐛 - 修复 bug
-   ✨ - 引入新功能
-   🍎 - 修复 MacOS 下的问题
-   📝 - 写文档
-   🚀 - 部署新功能
-   ✅ - 添加测试用例
-   🔖 - 发版/版本标签
-   🔒 - 修复安全问题
-   🐧 - 修复 Linux 下的问题
-   🚨 - 移除 linter 的警告
-   🚧 - 工作在进行中
-   💚 - 修复 CI 构建问题
-   ⬇️ - 降级依赖库
-   🏁 - 修复 Windows 下的问题
-   ⬆️ - 升级依赖库
-   👷 - 添加 CI 构建系统
-   🔧 - 改变配置文件
-   🔨 - 大重构
-   🎉 - 初次提交
-   💄 - 升级 UI 和样式文件
